# java-ee-online-store
Full-stack Java EE web application for an online store built with JSP, Servlets and the MVC architecture. Features user authentication, shopping cart functionality, order processing, and database persistence with MySQL.

# Java EE Online Store

This project is a Java EE web application that implements a simple online store following the MVC architecture.

## Technologies
- Java
- JSP
- Servlets
- Maven
- MySQL / MariaDB
- Apache Tomcat

## Features
- User registration and login
- Shopping cart management
- Order processing
- Order history and cancellation
- Database persistence

## Architecture
The application follows the MVC pattern:
- **Model:** JavaBeans and database entities
- **View:** JSP pages
- **Controller:** Java Servlets
